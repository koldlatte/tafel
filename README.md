# Tafel

## Getting Started

```bash
# Install and link
npm i
npm run build
npm link
```

## Commands

### `ls`

List all open issues. Add `all` to list all issues (or `la`).

```bash
# list open issues
tafel ls
tafel list

# list all issues
tafel ls all
tafel list all
tafel la
```

### `create <name>`

Create a new issue with given name.

```bash
tafel create "my first issue"
```
